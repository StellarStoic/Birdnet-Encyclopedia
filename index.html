Featheration sign in

Nostr lets you use one identity across compatible apps. Birds.name will use it later for optional federation features around your bird observations.

Your private key is your account.

Anyone who gets your private key can post, edit, or delete as you in Nostr apps. Birds.name generates the key in this browser and does not upload it. To write events safely, put the private key in a signer extension or app and let Birds.name ask that signer for signatures.

Recommended setup

  1. Create a Nostr key here.
  2. Copy the private key and save it somewhere safe, such as a password manager. If you lose it, the account cannot be recovered and you will need to create a new one.
  3. Install a signer app, add your private key (nsec1...), and use it to connect with Birds.name so you can publish and interact with other members in the Featherverse without storing your private key here.

Relays

Relays are like servers that receive, store, and share events. Birds.name uses relays to publish your diary events and to find them again when you log in with the same Nostr identity.

Personal relays are used first for publishing and reading Birds.name events. App default relays are optional fallbacks.

About Birds.name

Birds.name is a multilingual bird-name guide built around BirdNET-Pi translation files, Wikipedia descriptions, IUCN conservation data, bird media, and personal observation notes.

Use it to search birds by local or scientific name, compare names across languages, check conservation status, keep a private diary of birds you observed, and explore BirdNET-Pi or BirdWeather detection statistics.

Translation help needed

Some languages still need better interface translations and bird-name corrections. If you can help, go to the GitHub repository and open a pull request. Do not know how? Ask AI to guide you through making a PR.
